DEMO中演示redux以及redux-devtools的使用步骤
安装 npm install --save redux-thunk redux-devtools-extension

## Available Scripts

In the project directory, you can run:

### `yarn start`
本地运行项目

### `yarn build`
项目打包

### `yarn eject`
暴露webpack配置
**Note: this is a one-way operation. Once you `eject`, you can’t go back!**
